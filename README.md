# mojo-crypto

Cryptographic primitives implemented in [Mojo](https://www.modular.com/mojo).

## Requirements

- [pixi](https://pixi.sh) — manages the Mojo toolchain and dependencies

## Commands

```bash
pixi run fmt
pixi run test
pixi run bench
pixi run bench-rust
```
